Privacy Policy for JLPT Quiz

Last Updated: Feb, 22 2025

Introduction

Welcome to JLPT Quiz. We are committed to protecting your privacy and ensuring you have a positive experience on our app. This privacy policy outlines our practices for collecting, using, maintaining, and disclosing information collected from users of the JLPT Quiz application.

Information We Collect

Account Information

  • Email address (for registered users)
  • Authentication tokens
  • Account preferences
  • Quiz progress and scores

Usage Data

  • Quiz completion statistics
  • Study patterns
  • Feature usage analytics
  • App performance metrics
  • Device information (OS version, device type)

Technical Data

  • IP address
  • Device identifiers
  • System logs
  • Performance data
  • Error reports

How We Use Your Information

Essential Operations

  • User authentication and account management
  • Progress tracking and score maintenance
  • Quiz state preservation
  • Performance optimization

App Improvement

  • Feature enhancement based on usage patterns
  • Bug identification and resolution
  • User experience optimization
  • Content refinement

Communication

  • Account-related notifications
  • Password reset communications
  • Important updates about the app
  • Response to user inquiries

Data Storage and Security

Storage Location

  • User data is stored securely on Supabase servers
  • Local device storage for offline functionality
  • Temporary cache for performance optimization

Security Measures

  • End-to-end encryption for data transmission
  • Secure authentication protocols
  • Regular security audits
  • Automated threat detection
  • Regular backup procedures

Data Sharing and Disclosure

We do not sell, trade, or rent users' personal identification information to others. We may share generic aggregated demographic information not linked to any personal identification information regarding visitors and users with our business partners and trusted affiliates for the purposes outlined above.

Third-Party Services

We use the following third-party services:

  • Supabase for user authentication and data storage
  • Analytics tools for app performance monitoring
  • Error tracking services for bug resolution

User Rights and Controls

You have the right to:

  • Access your personal data
  • Correct inaccurate data
  • Request data deletion
  • Export your data
  • Opt-out of non-essential data collection
  • Choose between guest and registered user modes

Data Retention

  • Account information: Retained until account deletion
  • Quiz progress: Stored until user requests deletion
  • Usage data: Retained for 12 months
  • Technical logs: Kept for 30 days
  • Anonymous data: May be retained indefinitely

Children's Privacy

The JLPT Quiz app does not knowingly collect personal information from children under 13. If you are under 13, please use the app with parental supervision.

Changes to Privacy Policy

We reserve the right to update this privacy policy at any time. Users will be notified of any material changes via:

  • In-app notifications
  • Email (for registered users)
  • App store update notes

Guest Mode

Users can access the app in guest mode with:

  • Limited data collection
  • No personal information storage
  • Temporary session data only
  • No cross-device synchronization

Offline Usage

When using the app offline:

  • Data is stored locally on the device
  • Synced with our servers when online
  • Protected with local encryption
  • Cleared upon app uninstallation

Deletion Requests

To request account deletion:

  1. Go to Settings > Privacy
  2. Select "Delete Account"
  3. Confirm deletion request
  4. Receive confirmation email

All personal data will be permanently deleted within 30 days of request.

We aim to respond to all inquiries within 48 hours.
